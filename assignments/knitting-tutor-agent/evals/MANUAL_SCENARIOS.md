# KnitCoach 수동 테스트 시나리오

서버는 `OPENAI_API_KEY=`와 `KNITCOACH_ENABLE_VISION=0`으로 실행합니다.
각 시나리오는 **새 뜨개 대화**에서 시작해 입력을 그대로 복사합니다.

## 1. 완전 초보의 키링 목표

선택: `처음부터 배우기`

입력:

> 뜨개질은 처음인데 키링을 만들어보고 싶어요

기대:

- 키링을 첫 작품으로 인식한다.
- 코바늘 소품을 추천한다.
- 납작한 모양과 솜을 넣은 캐릭터의 난이도 차이를 설명한다.
- 원하는 크기와 보유 실·바늘을 먼저 질문한다.
- `짧게 연습해 보기`나 임의 도안을 바로 제시하지 않는다.

## 2. 목표가 없는 완전 초보

선택: `처음부터 배우기`

입력:

> 뜨개질을 아무것도 모르는데 시작하고 싶어요

기대:

- 코바늘과 대바늘의 차이를 결과물 중심으로 설명한다.
- 만들고 싶은 것, 보유 도구, 가능한 시간을 질문한다.
- 준비 정보를 받기 전에 연습 과제를 만들지 않는다.

## 3. 다음 기법으로 이어가기

선택: `배운 내용 이어가기`

입력:

> 지난번 사슬뜨기 연습 다음 단계가 궁금해요

기대:

- 사슬뜨기를 처음부터 반복하지 않고 짧은뜨기로 연결한다.
- 연습 분량과 성공 확인 기준을 알려준다.
- 마커, 장력 같은 용어가 나오면 뜻을 설명한다.

## 4. 연습 결과 점검

선택: `배운 내용 이어가기`

입력:

> 짧은뜨기 연습 결과를 이야기할게요. 시작은 12코였는데 마지막에는 14코가 됐어요

기대:

- 결과를 무시하고 새 기법을 가르치지 않는다.
- 첫 코·마지막 코와 단 시작 위치를 확인하게 한다.
- 한 번에 고칠 행동 하나를 제안한다.

## 5. 사진 없는 코스터 진단

선택: `작품 문제 해결하기`

입력:

> 코스터 가장자리가 자꾸 휘어요. 왜 그런가요?

기대:

- 사진에서 확인했다고 말하지 않는다.
- 코 수, 단 끝 코 누락, 장력 차이를 원인 후보로 제시한다.
- 전체 사진과 문제 부위 사진을 요청한다.

## 6. 원형뜨기 코 증가 진단

선택: `작품 문제 해결하기`

입력:

> 원형뜨기 단 시작 위치에서 자꾸 코가 늘어나요

기대:

- 단 시작 코를 중복해서 떴을 가능성을 안내한다.
- 단 시작 위치에 표시 고리를 거는 방법을 제안한다.

## 7. 영문 도안 풀어 읽기

선택: `도안 풀어 읽기`

입력:

> ch 12, sc in each ch

기대:

- ch와 sc를 각각 사슬뜨기와 짧은뜨기로 설명한다.
- 원문에 없는 반복 단 수를 만들지 않는다.

## 8. 설명을 도안으로 변환

선택: `도안 풀어 읽기`

입력:

> 사슬 12코로 시작해서 짧은뜨기 5단을 도안으로 바꿔줘

기대:

- 시작 사슬 12코와 짧은뜨기 1–5단을 구분한다.
- 입력에 없는 기둥코와 마무리 방식은 확정하지 않는다.

## 9. 가진 대바늘과 실로 작품 찾기

선택: `가진 재료로 시작하기`

입력:

> 울 실 두 타래와 5mm 대바늘로 뭘 만들 수 있을까요?

기대:

- 실 굵기와 총 길이를 확인해야 한다고 알린다.
- 조건부로 짧은 목도리나 헤어밴드를 추천한다.
- 보지 못한 실 굵기를 단정하지 않는다.

## 10. 실 라벨 질문

선택: `가진 재료로 시작하기`

입력:

> 이 실 라벨에 맞는 바늘과 부자재를 알려줘

기대:

- 라벨 사진이나 권장 바늘 크기, 실 길이를 요청한다.
- 마커와 돗바늘의 용도를 설명한다.

## 11. 사진 없이 사진 도안을 요구

선택: `사진 속 작품 만들기`

입력:

> 사진 속 가방과 똑같은 도안을 만들어줘

기대:

- 사진 첨부나 오프라인 샘플 선택을 요구한다.
- 보지 않은 작품의 구조나 코 수를 추측하지 않는다.

## 12. 오프라인 샘플 가방

선택: `사진 속 작품 만들기`

작업:

1. `네트 숄더백 (샘플·오프라인)`을 선택한다.
2. 기본값으로 도안 초안을 만든다.

기대:

- API 호출 없이 도안 카드가 생성된다.
- 가방바닥, 가방몸통, 손잡이, 크로스끈 구조가 표시된다.
- 게이지 샘플과 사진에서 확정할 수 없는 가정을 구분한다.

## 13. 초보 가방 커리큘럼과 진도

선택: `처음부터 배우기`

같은 대화에서 아래 문장을 차례대로 입력합니다.

1. `뜨개질은 처음인데 작은 가방을 만들어보고 싶어요`
2. `코바늘 납작 미니 토트백을 선택할게요`
3. `사슬뜨기 연습 완료했어요`
4. `짧은뜨기 연습 완료했어요`
5. `빼뜨기 연습 완료했어요`
6. `손잡이 연습 완료했어요`

기대:

- 첫 응답은 코바늘과 대바늘의 차이를 설명하고 세 가지 초보 가방을 제안한다.
- 가방을 선택하기 전에는 임의의 `짧게 연습해 보기`가 나오지 않는다.
- 선택 후 사슬뜨기부터 한 기법씩 연습하며 성공 기준과 `연습 완료` 안내가 나온다.
- 왼쪽 `MY LEARNING PATH`의 진도가 0/4부터 4/4까지 증가한다.
- 마지막 완료 후 연습 대신 사슬 21코부터 시작하는 실제 미니 토트백 제작 가이드가 열린다.
